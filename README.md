# Courier Management System

## 📌 Description
A Java-based application to manage courier services including booking, tracking, and delivery.

## 🛠 Technologies Used
- Java
- OOP Concepts

## 🚀 Features
- Add courier
- Track shipment
- Manage delivery status

## 📂 How to Run
1. Clone the repo
2. Open in IntelliJ / Eclipse
3. Run main class
